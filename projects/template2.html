Mechanism design for collaborative learning with strategic agents.

Motivation

Imagine a world where many intelligent agents—like robots, apps, or recommendation systems—are all trying to learn how to make better decisions by interacting with the same environment. Each one explores and observes the environment on its own, and share what they learn to help each other improve—self-driving cars learning from each other's experiences, mobile apps adapting to users’ preferences by pooling data, or online platforms recommending products more effectively by collaborating across different user groups. This project is about designing and understanding such collaborative learning systems.
However, when agents are independent and have different capabilities, it’s not obvious how or why they would choose to cooperate—or how to fairly split the benefits. This project develops a way to analyze such scenarios through the lens of game theory.

How should fruits of labour be shared when contributions differ?

When autonomous agents interact amongst themselves, how does rationality impact collaboration?

Results / Contributions

We ask and try to answer the following questions: Which groups should form? Will everyone want to work together? How should the benefits of collaboration be shared? By using ideas from game theory and machine learning, we show that collaboration can be stable, beneficial, and fair—even when agents are quite different. Our results offer insights for building more cooperative, efficient, and intelligent systems.
